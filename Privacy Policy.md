# 星月英语隐私政策

最后更新：2026-04-26

星月英语是一款本地优先的英语学习 App。应用的核心数据包括导入的学习素材、字幕、词库、练习记录、复习记录、收藏/错词/已掌握状态、设置项和可选的云端服务密钥。

## 我们处理哪些数据

- 学习数据：单词、短语、文章、字幕、点词保存记录、练习结果、FSRS 复习进度。
- 导入素材：用户主动选择或分享给应用的 TXT、PDF、SRT、ASS、VTT、JSON、APKG、音频、视频和网页链接。
- 词典数据：内置词典、用户导入词典、查询历史和收藏状态。
- 设置数据：每日学习数量、TTS 参数、语速、音量、云端服务区域、可选的云端 API Key。
- 缓存数据：TTS 生成的音频缓存、导入处理产生的临时文件、媒体/字幕处理缓存。

## 本地存储

上述数据默认保存在设备本机的应用私有目录和本地数据库中。应用不会在未触发对应功能时主动上传这些数据。

用户可以通过应用内导出功能生成备份文件。备份文件可能包含学习记录、词库、字幕、导入来源、设置和本地数据库内容。请只把备份文件保存到可信位置。

## 文件访问

应用通过 Android 系统文件选择器或系统分享面板接收文件。应用只读取用户主动选择、分享或授权的文件，不扫描设备上的其他文件。

对需要长期学习的素材，应用可能会保存文件 URI 授权，或将媒体/字幕复制到应用私有目录，便于离线播放、字幕时间轴、点词保存和复习。

## 网络访问

应用申请 `INTERNET` 权限，网络访问只用于用户主动触发或配置的功能：

- 云端 ASR：当用户配置云端密钥并请求自动生成字幕时，应用会把待识别音频或可访问的音频地址发送到所选云端服务。
- 云端 TTS：当用户配置云端密钥并播放云端发音时，应用会把待合成文本、音色、语速和音量参数发送到所选云端服务，并把返回音频缓存在本机。
- 云端词典/翻译：当用户配置云端密钥并请求云端释义、翻译或连接测试时，应用会发送对应单词、句子或测试请求。
- 平台链接导入：当用户导入 YouTube、Bilibili、抖音等链接时，应用会请求该链接及相关媒体/字幕资源，用于生成学习素材。
- 直接 URL 导入：当用户输入或分享 HTTP/HTTPS 链接时，应用会访问该链接以获取文本、字幕、音频或视频。

未配置云端密钥时，云端 ASR、云端 TTS 和云端词典不会执行上传请求，并会显示需要配置的状态。

## 第三方服务

应用当前可能访问以下第三方服务或资源，具体取决于用户启用的功能：

- 阿里云百炼 / DashScope：用于可选的云端语音识别、语音合成、词典释义和翻译。
- 用户导入的平台链接所属网站：用于解析用户主动提供的网页、音频、视频或字幕资源。

第三方服务会按照其自身条款处理请求内容。请不要把不希望发送到第三方的私人音频、视频、文本或密钥用于云端功能。

## 权限说明

当前最终 APK 中声明的系统权限：

- `android.permission.INTERNET`：用于云端 ASR/TTS/词典、平台链接导入和直接 URL 导入。
- `android.permission.ACCESS_NETWORK_STATE`：由网络组件合入，用于判断当前网络状态，便于显示连接失败或重试状态。

最终 APK 还包含应用内部的动态广播保护权限 `com.xingyue.english.DYNAMIC_RECEIVER_NOT_EXPORTED_PERMISSION`。它只用于限制应用内部动态广播接收器，不授予应用读取用户个人数据的能力。

当前未申请：

- 相机权限
- 麦克风权限
- 通讯录权限
- 精确或粗略位置权限
- 短信/通话记录权限
- 全盘文件读取权限

## 数据删除

用户可以通过系统“应用信息 -> 存储 -> 清除数据”删除本机数据库、缓存、导入素材和设置。应用内导出的备份文件不属于应用私有数据，需由用户在保存位置自行删除。

## 安全说明

应用不会把用户配置的云端 API Key 写入截图或测试报告。密钥保存在本机应用数据库中，用于用户主动启用的云端功能。

正式版本默认不启用 Android 系统自动备份，避免学习数据、导入素材或云端密钥被系统备份到外部账号。用户需要备份时，应使用应用内导出功能。

## 儿童隐私

应用没有面向儿童收集个人信息的功能。如果未成年人使用云端功能，应由监护人确认导入内容和第三方服务配置。

## 联系与变更

后续如果新增权限、账号系统、在线同步、广告 SDK、统计 SDK 或新的第三方服务，本政策应在发布前更新，并在应用内提供对应说明。
